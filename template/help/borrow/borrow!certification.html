  • 必要认证资料和可选认证资料有哪些？

    认证项如下：（点击某项认证，查看详细说明）
    必要认证资料
    工薪贷
    身份认证、个人信用报告、劳动合同或在职证明、近3个月工资卡银行流水
    生意贷
    身份认证、个人信用报告、经营证明、近6个月常用银行卡流水
    网商贷
    身份认证、提供网店地址及阿里旺旺账号、需进行QQ视频审核
    可选认证资料
    房产认证、购车认证、结婚认证、学历认证、技术职称认证、手机实名认证、微博认证、居住地证明
    （点击各项认证，链接至各项认证说明处）

  • 身份认证

    您上传的身份证照片需和您绑定的身份证一致，否则将无法通过认证。
    请同时提供下面两项资料：
    1. 本人身份证原件的正、反两面照片。
    2. 本人手持身份证正面头部照，（确保身份证上的信息没有被遮挡，避免证件与头部重叠）。
    认证示例：
    身份证正反面照片
    本人手持身份证正面头部照片
    认证有效期：
    永久

  • 信用报告

    个人信用报告是由中国人民银行出具，全面记录个人信用活动，反映个人信用基本状况的文件。本报告是多网了解您信用状况的一个重要参考资料。
    认证说明：
    个人信用报告需15日内开具。
    认证条件：
    信用记录良好。
    认证有效期：
    6个月
    认证示例：
    个人信用报告
    如何办理：
    可去当地人民银行打印，部分地区可登录个人信用信息服务平台。
    全国各地征信中心联系方式查询
    个人信用信息服务平台
    关于个人查询本人信用报告实施收费的公告

  • （工薪贷）工作认证

    您的工作状况是多网评估您信用状况的主要依据之一。
    认证说明：
    请提供下面任意一项资料：
    1. 加盖单位公章（或劳动合同专用章）的劳动合同。
    2. 最近1个月内开具的加盖单位公章（或人力章、财务章）的机打（手写无效）在职证明。
    3. 带有姓名、照片、工作单位名称的工作证。
    认证条件：
    本人需在现单位工作满3个月。
    认证有效期：
    6个月
    认证示例：
    劳动合同
    在职证明
    工作证

  • （生意贷）工作认证

    您的工作状况是多网评估您信用状况的主要依据之一。
    认证说明：
    请同时提供下面两项资料：
    1. 注册满1年的营业执照正、副本。
    2. 经营场地租赁合同＋90天内的租金发票或水电单据。
    认证条件：
    本人名下的企业经营时间需满1年。
    认证有效期：
    6个月
    认证示例：
    营业执照正、副本
    租赁合同和水电发票

  • （网商贷）工作认证

    您的工作状况是多网评估您信用状况的主要依据之一。
    认证说明：
    网商用户无需上传任何工作认证相关资料，但需进行视频审核，多网客服会尽快主动联络您进行视频审核。在审核过程中，需远程展示以下资料：
    1. 淘宝/天猫近3个月交易明细或支付宝近12个月账单。
    2. 淘宝/天猫的支付宝实名认证页面。
    3. 淘宝/天猫信用以及订单借款账户页面。
    4. 淘宝/天猫处罚积分页面。
    认证条件：
    本人名下的店铺需在淘宝或天猫经营满半年，且近期无淘宝严重处罚，店铺等级达到2钻, 好评率达到95%，动态评分达到4.5，无法估价产品和虚拟产品占比不超过40%。近3个月交易总额满3万，且交易笔数超过50笔。
    认证有效期：
    6个月

  • （工薪贷）收入认证

    您的银行流水单是多网评估您收入状况的主要依据之一。
    认证说明：
    请提供下面任意一项资料：
    1. 可体现工资项的最近3个月的工资卡银行流水单。
    2. 可体现工资项的最近3个月的网银电脑截屏。
    认证条件：
    本人名下近3个月的月收入均在2000以上。
    认证有效期：
    6个月
    认证示例：
    银行卡流水
    网银电脑截图

  • （生意贷）收入认证

    您的银行流水单是多网评估您收入状况的主要依据之一。
    认证说明：
    请提供下面任意一项资料：
    1. 可体现经营情况的最近6个月的个人银行卡流水单，或网银电脑截屏。
    2. 可体现经营情况的最近6个月的企业银行卡流水单，或网银电脑截屏。
    认证条件：
    本人提交的流水可反应真实有效的经营情况。
    认证有效期：
    6个月
    认证示例：
    银行卡流水
    网银电脑截图

  • 房产认证

    房产认证是证明借入者资产及还款能力的重要凭证之一。
    认证说明：
    请提供下面任意一项资料：
    1. 房屋产权证明。
    2. 购房合同 + 近3个月的还贷流水。
    3. 购房发票 + 近3个月的还贷流水。
    4. 按揭合同 + 近3个月的还贷流水。
    认证条件：
    必须是商品房，且房产是本人名下所有或共有的。
    认证有效期：
    永久
    认证示例：
    房产证
    购房发票
    购房合同

  • 购车认证

    购车认证是证明借入者资产及还款能力的重要凭证之一。
    认证说明：
    请同时提供下面两项资料：
    1. 车辆行驶证的原件照片。
    2. 本人和车辆的合影（照片需露出车牌号码）。
    认证条件：
    车辆必须是本人名下所有。
    认证有效期：
    永久
    认证示例：
    车辆行驶证
    与车辆合影

  • 结婚认证

    借入者婚姻状况的稳定性，是多网考核借款人信用的评估因素之一。
    认证说明：
    请同时提供下面三项资料：
    1. 结婚证书的原件照片。
    2. 配偶身份证原件的正、反两面照片。
    3. 本人和配偶的近照合影一张。
    认证有效期：
    永久
    认证示例：
    结婚证

  • 学历认证

    借出者在选择借款申请投标时，借入者的学历也是一个重要的参考因素。为了让借出者更好、更快地相信您的学历是真实的，建议您对学历进行在线验证。
    认证说明：
    2001年至今获得学历，需学历证书编号：
    1. 访问学信网，找到“零散查询”信息，点击【查询】；
    2. 输入证书编号、姓名、查询码（通过手机短信获得，为12位学历查询码）、验证码，进行查询；
    3. 查询成功后，您将查获得《教育部学历证书电子注册备案表》；
    4. 将该表右下角的12位在线验证码（见学历认证1）输入文本框，点击【提交审核】。
    1991年至今获得学历，无需学历证书编号：
    1. 访问学信网，找到“本人查询”信息，点击【查询】；
    2. 注册学信网账号，登录学信网，访问【学历信息】页面；
    3. 选择您的最高学历，点击【申请验证报告】（通过手机短信获得12位学历查询码，此查询码与多网所需验证码不同）；
    4. 申请成功后，您将获得12位在线验证码（见学历认证2）；
    5. 将12位在线验证码输入文本框，点击【提交审核】。
    认证条件：
    大专或以上学历（普通全日制）。
    认证有效期：
    永久
    认证示例：
    学历认证1
    学历认证2

  • 技术职称认证

    技术职称是经专家评审、反映一个人专业技术水平并作为聘任专业技术职务依据的一种资格。不与工资挂钩，是多网考核借款人信用的评估因素之一。
    认证说明：
    技术职称证书的原件照片。
    认证条件：
    国家承认的二级及以上等级证书。例如律师证、会计证、工程师证等。
    认证有效期：
    永久

  • 手机实名认证

    手机实名认证需要用户上传手机流水单。手机流水单是用户最近一段时间内的详细通话记录，是多网用以验证借入者真实性的重要凭证之一。
    认证说明：
    绑定的手机号码最近3个月的手机详单原件的照片。如详单数量较多可分月打印并上传每月前5日部分（每月详单均需清晰显示机主手机号码）。
    认证有效期：
    永久
    如何办理：
    前往最近的手机运营商营业厅，提供手机服务密码或机主身份证明即可打印。

  • 微博认证

    随着网络时代的发展，网络信息已成为了解借入者真实性的最新渠道之一。多网与时俱进，将借入者网络关系评估，作为评估借入者的信用的一种方式。
    认证说明：
    1. 成为多网粉丝：
    请点击链接http://e.weibo.com/duonet，把多网加为关注
    请点击链接http://t.qq.com/duonet，把多网加为关注；
    2. 发送验证信息
    在微博上，转发一条最新的多网的微博，并用“私信”功能，发送您的多网昵称给多网；
    3. 在输入框输入您的新浪微博昵称输入文本框，点击【提交审核】。
    认证条件：
    50粉丝 + 50微博（目前仅支持新浪微博）
    认证有效期：
    永久

  • 居住地证明

    居住地的稳定性，是多网考核借款人的主要评估因素之一。
    认证说明：
    请提供下面任意一项资料：
    1. 用本人姓名登记的水、电、气最近3个月缴费单。
    2. 用本人姓名登记的固定电话最近3个月缴费单。
    3. 本人的信用卡最近2个月的月结单。
    4. 本人的自有房产证明。
    5. 本人父母的房产证明，及证明本人和父母关系的证明材料。
    认证有效期：
    6个月

  • （工薪贷）工作单位的注册地必须在中国大陆吗？

    多网网站支持的用户是中国大陆地区公民，且工作单位或经营场所注册地必须在中国大陆，港澳台地区目前不支持。

  • （工薪贷）工资发现金，没有发到银行卡上怎么办？

    如果您无法提供工资卡流水，也可提供能体现您收支情况的常用银行卡流水。

  • （生意贷）没有经营地证明怎么办？

    经营地证明中的经营场地租赁合同是您必须要提供的，90天内的租金发票或水电单据您可选择其中一项提供，如果您的水电单据存在特殊情况时，客户经理将致电向您了解具体情况并沟通解决办法。

  • （生意贷）借款人不是法人本人，可以申请吗？

    生意贷产品要求申请人限于法人代表。如遇申请人身份特殊时，客户经理将致电向您了解具体情况并沟通解决办法。

  • （生意贷）借款人是法人本人，但银行流水在别人名下，可以申请吗？

    生意贷产品要求提供法人名下或企业对公银行流水。如遇申请人情况特殊时，客户经理将致电向您了解具体情况并沟通解决办法。

  • （网商贷）借款人不是淘宝或天猫商户，可以申请吗？

    目前多网网站暂不支持非淘宝或天猫的网商申请借款，要求至少在淘宝或天猫经营网店满半年。

  • “完整的银行流水，中间不能出现断层”是什么意思？

    要求银行流水有连续月份，内容包括交易日期、交易金额、余额、交易地点、交易摘要等。（点击银行卡流水，查看示例图片）

  • 如何上传、补充、修改或撤销资料？

    1.登陆多网网站，打开【我要借款】--【工薪贷】查看详情页面（以工薪贷为例），点击【申请借款】，【填写借款申请】后，您可以在【填写借款信息】的过程中上传【必要认证资料】；
    2.您也可以在【上传资料】页面中上传所有【必要上传资料】和【可选上传资料】。请您确保所有资料上传完整后点击【提交申请】，【提交申请】后您将无法补充上传资料。
    查看操作流程>>
    确认【提交申请】后，我们的客户经理将检查您的资料，如果您的资料上传有误或未上传齐全，我们将驳回您的资料，您可继续操作补充资料（建议您一次性将资料补充完整）。
    资料上传后，无法操作修改和撤销，您可按照上述说明继续补充上传。
