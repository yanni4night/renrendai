借款费用

  • 借款服务费

    人人贷收取的借款服务费将全部存于风险备用金账户用于人人贷的本金保障计划。
    服务费将按照借款人的信用等级来收取：
    信用等级AAABCDEHR
    服务费率0%1%1.5%2%2.5%3%5%

  • 借款管理费

    人人贷将按照借款人的借款期限，每月向借款人收取其借款本金的0.3%作为借款管理费。

  • 充值提现费用

    充值/提现时由第三方支付收取的支付费用。
    充值费用
    第三方支付平台将收取您充值金额的0.5%作为转账费用。扣除手续费的上限为100元。超过100元的部分将由人人贷承担。
    在充值时，用户也可以使用免费充值券，抵去该笔充值费用。如何获得免费充值券？
    提现费用
    金额2万元以下2万(含)-5万元5万(含)-100万元
    手续费1元/笔3元/笔5元/笔

  • 逾期罚息

    当用户的借款发生逾期时，正常利息停止计算。按照下面公式计算罚息：
    : 罚息总额
    : 逾期本息
    : 对应罚息利率
    : 逾期天数
    逾期天数1—30天 31天以上
    罚息利率0.05%0.1%

  • 逾期管理费

    用户的借款发生逾期时，正常借款管理费用停止计算。按照下面公式计算：
    : 逾期管理费
    : 逾期本息
    : 逾期管理费率
    : 逾期天数
    逾期天数1-30天31天以上
    逾期管理费率0.1%0.5%

  • 人人贷年利率范围

    人人贷目前的利率范围为10%-24%。在人人贷平台上借贷的最高年利率设定为同期银行借款年利率的4倍。且随着银行借款利率的调整，人人贷的利率上限也将随之调整。
    注：
    1. 人人贷的利率的调整会在商业银行借款年利率调整后1个月内进行调整。
    2. 在利率调整之前成功的借款不受调整的影响。

如何申请

  • 借款申请条件有哪些？是否需要担保和抵押？

    不同产品的申请人需要符合不同的申请条件。查看详情>>
    您通过人人贷网站申请借款不需要抵押和担保。

  • 借款申请步骤有哪些？

    1. 填写借款申请
    2. 填写个人信息和上传认证资料
    3. 通过审核
    4. 筹集借款
    5. 获得借款

  • 我能获得多少借款金额？

    您可申请的借款额度范围是3,000-500,000（请您按照实际需求填写）。
    您可获得的借款额度将在信审部门审核完成后，由您的整体资质决定（可能与您申请的借款金额不一致）。

  • 我能选择哪些借款期限和还款方式？

    目前您可选择的借款期限有3个月、6个月、9个月、12个月、15个月、18个月、24个月、36个月。
    人人贷网站的还款方式为等额本息，即每个月还相同金额的本金加利息。（点击借款计算器设置并计算，查看还款明细）

  • 如何填写借款申请表？

    1.登陆人人贷网站，打开【我要借款】--【工薪贷】查看详情页面（以工薪贷为例），点击【申请借款】；
    2.进入第①步【填写借款申请】,找到“借款申请信息填写”，输入借款标题、借款用途、借款金额、借款期限、年化利率、借款描述，点击【保存并继续】，进行借款申请。查看操作流程>>

  • 如何修改或取消借款申请？

    在您发布借款申请后，无法自行修改或取消申请，请您在发布时注意认真填写各项信息。如遇特殊情况需修改或取消申请时，您可以联系客服寻求帮助（400-027-8080）。

    工薪贷

    生意贷

  • 身份认证

    您上传的身份证照片需和您绑定的身份证一致，否则将无法通过认证。
    请同时提供下面两项资料：
    1. 本人身份证原件的正、反两面照片。
    2. 本人手持身份证正面头部照，（确保身份证上的信息没有被遮挡，避免证件与头部重叠）。
    认证示例：
    身份证正反面照片
    本人手持身份证正面头部照片
    认证有效期：
    永久

    认证说明：
    个人信用报告需15日内开具。
    认证条件：
    信用记录良好。
    认证有效期：
    6个月
    认证示例：
    个人信用报告
    如何办理：
    可去当地人民银行打印，部分地区可登录个人信用信息服务平台。
    全国各地征信中心联系方式查询
    个人信用信息服务平台
    关于个人查询本人信用报告实施收费的公告

    借入者婚姻状况的稳定性，是人人贷考核借款人信用的评估因素之一。
    认证说明：
    请同时提供下面三项资料：
    1. 结婚证书的原件照片。
    2. 配偶身份证原件的正、反两面照片。
    3. 本人和配偶的近照合影一张。
    认证有效期：
    永久
    认证示例：
    结婚证

  • 学历认证

    借出者在选择借款申请投标时，借入者的学历也是一个重要的参考因素。为了让借出者更好、更快地相信您的学历是真实的，建议您对学历进行在线验证。
    认证说明：
    2001年至今获得学历，需学历证书编号：
    1. 访问学信网，找到“零散查询”信息，点击【查询】；
    2. 输入证书编号、姓名、查询码（通过手机短信获得，为12位学历查询码）、验证码，进行查询；
    3. 查询成功后，您将查获得《教育部学历证书电子注册备案表》；
    4. 将该表右下角的12位在线验证码（见学历认证1）输入文本框，点击【提交审核】。
    1991年至今获得学历，无需学历证书编号：
    1. 访问学信网，找到“本人查询”信息，点击【查询】；
    2. 注册学信网账号，登录学信网，访问【学历信息】页面；
    3. 选择您的最高学历，点击【申请验证报告】（通过手机短信获得12位学历查询码，此查询码与人人贷所需验证码不同）；
    4. 申请成功后，您将获得12位在线验证码（见学历认证2）；
    5. 将12位在线验证码输入文本框，点击【提交审核】。
    认证条件：
    大专或以上学历（普通全日制）。
    认证有效期：
    永久
    认证示例：
    学历认证1
    学历认证2

  • 技术职称认证

    技术职称是经专家评审、反映一个人专业技术水平并作为聘任专业技术职务依据的一种资格。不与工资挂钩，是人人贷考核借款人信用的评估因素之一。
    认证说明：
    技术职称证书的原件照片。
    认证条件：
    国家承认的二级及以上等级证书。例如律师证、会计证、工程师证等。
    认证有效期：
    永久

  • 手机实名认证

    手机实名认证需要用户上传手机流水单。手机流水单是用户最近一段时间内的详细通话记录，是人人贷用以验证借入者真实性的重要凭证之一。
    认证说明：
    绑定的手机号码最近3个月的手机详单原件的照片。如详单数量较多可分月打印并上传每月前5日部分（每月详单均需清晰显示机主手机号码）。
    认证有效期：
    永久
    如何办理：
    前往最近的手机运营商营业厅，提供手机服务密码或机主身份证明即可打印。

  • 微博认证

    随着网络时代的发展，网络信息已成为了解借入者真实性的最新渠道之一。人人贷与时俱进，将借入者网络关系评估，作为评估借入者的信用的一种方式。

信用审核

  • 什么是信用审核？

    申请借款的用户需要根据不同的产品填写借款信息，包括个人信息、家庭信息、工作信息、资产信息、信用信息，并提交相应的信用认证材料。信审部门会综合评估借款人的个人、家庭、工作、资产、信用情况，最终根据借款人的整体信用资质给出相应的信用分数、信用等级及借款额度。

  • 提交认证资料后何时开始审核？审核时间多久？

    我们将在您点击【提交申请】后，确认您的必要认证资料已上传齐全，才会为您提交审核。
    您的借款申请将于3个工作日内为您审核完成。

  • 为何我提交一部分认证资料，但3个工作日后没有收到审核结果？

    请确认您的必要认证资料已上传完整，且已点击【提交申请】，才会进入审核。

  • 如何查看我的审核进度和结果？

    登陆人人贷网站，打开【我要借款】--【工薪贷】查看详情页面（以工薪贷为例），点击【申请借款】，【填写借款信息】并点击【提交申请】后，即可查看您的审核进度。查看操作流程>>
    审核完成后，如果您的申请通过，您将收到审核通过的站内信和邮件通知，同时客户经理将致电您沟通借款金额、期限和利率；
    如果您的申请未通过，您将收到申请驳回的短信、站内信和邮件通知，您可登陆人人贷账户或邮箱查看驳回原因。

  • 如何联系审核人员？

    您无需主动联系审核员，审核员在需要时会主动联系您。如遇特殊情况时，客户经理将致电与您沟通，您也可以联系客服寻求帮助（400-027-8080）。

  • 审核员会打电话给我的工作单位或亲属朋友吗？

    审核人员会根据您在申请表中填写的工作单位或亲属朋友联系人信息，进行电话核实。在致电您的亲属朋友联系人核实前，会先与您进行电话确认。

  • 审核过程主要看哪些方面？如何提高我的借款成功率？

    信审部门会综合评估您的整体资质，包括您的个人情况、家庭情况、工作情况、资产情况、信用情况等。我们建议您尽量全面的上传真实有效的必要认证资料和可选认证资料，以提高您的借款成功率。

  • 资料被驳回的原因是什么？可否重新认证？

    1. 资料上传不完整：您可以按照资料驳回的站内信和邮件通知，重新认证上传；
    2. 资料暂不符合认证要求：您的资料未能通过审核，无法再次认证。

  • 被拒贷或流标的原因是什么？

    可能的原因有：
    1. 您的年龄暂不符合借款要求（22-55周岁）。
    2. 您本人不符合申请人身份条件。
    3. 您的工作年限或经营时间不符合申请条件。
    4. 您的企业或网店不符合申请条件。
    5. 您的月收入或月交易额不符合申请条件。
    6. 您在审核过程中未配合完成补充资料或信息核实工作，或您主动放弃申请。
    7. 审核人员对您的身份信息、工作信息、信用记录、还款能力等进行审核后认为您目前综合评分不足，暂时不符合借款要求。

  • 审核失败后如何操作？何时能再次申请？

    审核失败后，您将收到资料驳回和借款申请驳回的短信、站内信和邮件通知，您可登陆人人贷账户或邮箱查看驳回原因。（点击这里了解可能导致审核失败的原因）
    如果您因个人信息或认证资料未提交齐全导致审核失败，您可随时再次发布借款申请；
    如果您因申请条件不符合要求或认证资料未通过审核，您可在半年后再次发布借款申请。

  • 审核成功后如何操作？

    审核成功后，您将收到审核通过的站内信和邮件通知，同时客户经理将致电您沟通借款金额、期限和利率，确认后帮助您进入筹款。

  • 申请资料能否退还给我？

    根据行业规范，对于用户申请过程中提交的信息和资料不予退还。
    人人贷严格遵守国家相关的法律法规，对用户的隐私信息进行保护。未经您的同意，人人贷不会向任何第三方公司、组织和个人披露您的个人信息、账户信息以及交易信息（法律法规另有规定的除外）。
    同时，人人贷公司内部也设有严格、完善的权限管理体系，以保证每一位内部员工都只能查看自己职责和权限之内的数据和信息。

信用等级与额度

  • 什么是信用分数和信用等级？

    用户的信用分数是在通过人人贷审核后获得的，信用等级由信用分数转化而来，每个信用等级都对应的信用分数范围，信用分数和信用等级是借款人的信用属性，也是理财人判断借款人违约风险的重要依据之一。通常来讲借款人信用等级越高，其违约率越低，相应的借款成功率越高。
    目前信用等级由高到低分为AA、A、B、C、D、E、HR，具体请参考信用等级的分数区间：
    信用等级HREDCBAAA
    分数区间0-99100-109110-119120-129130-144145-159160+

  • 什么是信用额度？

    用户的信用额度是在通过人人贷审核员对所提供材料的审核后获得的，既是借款人单笔借款的上限也是借款者累积尚未还清借款的上限。
    例如，如果一个借款人信用额度为5万元，则在没有其他借款的情况下，用户可以发布总额最高为5万元的借款请求。也可以分多次发布借款请求，但尚未还清借款（以整笔借款金额计算）的总额不得超过5万元。

  • 如何获得信用分数、信用等级与信用额度？

    请您申请借款并上传认证资料，审核完成后，您将获得相应的信用分数、信用等级和信用额度。

  • 如何查看我的信用分数、信用等级与信用额度？

    登陆人人贷网站，打开【我的人人贷】--【账户管理】--【认证信息】页面，即可查看您的信用分数、信用等级、信用额度，及您进行的各项信用认证的情况。查看操作流程>>

  • 如何提高我的信用等级（信用分数）？

    您可以通过提交【可选认证资料】来提高信用分数，另外您在人人贷网站上良好的还款记录也会增加您的信用分数。

  • 如何提高我的信用额度？

    提额申请资格说明

    提额申请办法

    您会在每月还款日的前三天以电子邮件、站内信的方式收到还款提醒，并在还款日的前一天接到客服人员的电话提醒。

  • 是否可以提前还款？如何操作？

    您可以随时进行提前还款。
    登陆人人贷网站，打开【我的人人贷】--【借款管理】--【我的借款】页面，点击【提前还款】进行提前还款操作（与还款操作类似）。
    提前还款的用户需要支付给理财人借款剩余本金的1%作为违约金，不用再支付后续的利息及管理费用。提前还款金额公式如下：
    : 提前还款应还金额
    : 剩余本金
    : 当期本息及账户管理费
    : 提前还款违约金
    注：提前归还部分借款不视为提前还款，仍需支付全部借款利息及账户管理费。

  • 是否可以申请延期还款？

    您不可以申请延期还款。您需每月按时还款，以免影响您的信用记录。
